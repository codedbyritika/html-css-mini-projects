# Gym Website Landing Page

A simple and responsive gym website landing page built using HTML and CSS.
This project focuses on creating a clean UI layout with a functional registration form and basic navigation.

---

## Features

* Fixed navigation bar with logo and menu items
* Hero section with background image
* User registration form with essential fields and transitions
* Button hover effects and basic styling

---

## Tech Stack

* HTML5
* CSS3 (Flexbox, positioning, basic animations)

## What I Learned

* Structuring a webpage using semantic HTML
* Using Flexbox for layout alignment
* Handling fixed navigation bars
* Creating and styling forms
* Managing full-screen sections using viewport units
* Improving UI with hover effects and spacing

---

## Limitations

* Not fully responsive for all screen sizes yet
* No backend functionality 
* No JavaScript interactivity
